# CS:GO Map Pool Website

Простой сайт для управления мап-пулом CS:GO с панелью администратора.

## Особенности

- Отображение карт с баннерами и описанием
- Простая аутентификация: логин `admin`, пароль `admin18`
- Панель управления для изменения статуса карт
- Добавление/удаление карт
- Адаптивный дизайн
- Все данные хранятся в localStorage браузера

## Использование

### Основные возможности:
1. **Главная страница:** Просмотр текущего мап-пула
2. **Панель управления:** Редактирование, добавление и удаление карт

### Данные для входа в панель управления:
- **Логин:** `admin`
- **Пароль:** `admin18`

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. В настройках репозитория перейдите в раздел "Pages"
4. Выберите ветку `main` и папку `/root`
5. Сохраните изменения

Сайт будет доступен по адресу: `https://ваш-логин.github.io/имя-репозитория/`

## Настройка картинок

Замените URL баннеров на реальные изображения карт CS:GO:
1. Dust II
2. Mirage  
3. Ancient
4. Anubis
5. Overpass

Для каждой карты укажите URL изображения в соответствующем поле.
